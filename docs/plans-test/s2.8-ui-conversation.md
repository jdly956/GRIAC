## 3. Session de bout en bout (au navigateur)

1. Coller une Feature (avec un `[HYPOTHÈSE À VALIDER]` dedans) → « Créer la session » → l'écran session s'ouvre, badge « 0 — Récupération de la Feature », l'hypothèse de la Feature apparaît **en attente**.
2. Envoyer un message → la réponse de l'assistant s'affiche dans le fil + panneau « Dernier échange » : **sources mobilisées** (A3), avertissements éventuels, hypothèses ajoutées.
3. Poser une **question documentaire libre** → réponse sourcée dans le même fil (A2).
4. « Oui — étape suivante » → le badge passe à « 1 — Interview » ; **les hypothèses restent en attente** (A8 visible à l'écran).
5. Confirmer une hypothèse via son bouton → son statut passe à `confirmee`.
6. Boutons Export : le CSV se télécharge (ou 409 lisible si aucune US rédigée).
