## 5. Similarité de bon sens (avant-goût E2)

```sql
select c2.section, c1.embedding <=> c2.embedding as distance
from chunks c1, chunks c2 where c1.id <> c2.id
order by c1.id, distance limit 5;
```

Attendu : des distances cosinus calculables (l'opérateur `<=>` fonctionne) ; les chunks des deux specs authentification (v1/v2) sont plus proches entre eux que du guide d'installation.
