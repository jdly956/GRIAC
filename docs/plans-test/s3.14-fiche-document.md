## 2. En réel (navigateur)

1. « Mes documents » : **chaque chemin est un lien** → cliquer sur un document **parsé**. Attendus sur la fiche :
   - badges statut/référence/doublon ; identité (taille, sha256, version détectée, projet suggéré) ;
   - « **Voir le résultat du parsing** » : le markdown produit par docling **rendu** (titres, tableaux) ;
   - « **Chunks — N, dont M vectorisé(s)** » : chaque chunk dépliable avec son fil de titres, ses tokens, son contenu exact et ✅/⏳ d'embedding.
2. Cliquer sur un document **OCR requis** ou **en échec** → le détail de l'erreur (`erreur_parsing`) s'affiche, zéro chunk avec l'explication.
3. Cliquer sur un **doublon** → « Doublon de : <chemin> ».
4. Cas dérivé disparu (pod recréé sans ré-indexation) : « Dérivé introuvable sur ce pod… » — la fiche reste consultable.
